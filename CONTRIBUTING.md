# Contributing Guidelines

Use of generative AI tools is generally not permitted unless explicitly approved by the project lead, and cited properly. Please follow the directory stucture outlined below when contributing new models or experiments.

When submitting code, please make sure you use a linter to clean it up and follow the existing coding style. All configs and path should be relative to the repository root.
